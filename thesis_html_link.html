Data analysis of the Zimbabwe global findex database 2017¶

by Rujeko Musarurwa¶

This database is a large dataset that contains 1000 individuals information with over 100 different variables of which I have chosen to focus on 10 relevant variables for my analysis. The aim of this exploration is to assess the data and any relationships that may exist between the different attributes with the main variable of interest being savings. The main feature in the dataset is the influence on savings by having a mobile money account.

Logistic Regression should be used because the response variable is categorical that can only predict two possible outcomes of saved or did not save.
